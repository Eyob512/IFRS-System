# IFRS SYSTEM

IFRS Systems
Basic IFRS standars implementation using Spring, Hibernate and Mysql

## Installation

`git clone https://github.com/eyob512/IFRSSystem.git`

`cd IFRSSystem`

`mvn clean install`

`cd target`

`java -jar IFRSSystem-0.0.1-SNAPSHOT.jar`
